# Nohdetabase

能楽の公演情報を体系化し、検索・分析を容易にするデジタルデータベース

##  概要
日本の伝統芸能である能楽の公演情報は、現在さまざまな媒体や公式サイトに分散しており、横断的な検索やデータとしての活用が難しいという課題があります。
このプロジェクトは、点在する公演情報を構造化されたデータとして集約し、伝統文化の記録とIT技術を繋ぐプラットフォームとして機能させることを目的に作成しました。

##  特徴
- 公演データの構造化: 演目、演者、流派、日程、開催場所などの情報を整理し、データベース化。
- 検索・アクセス性の向上: 複雑な条件から目的の公演情報を素早く見つけ出す機能。
- SEOを意識した情報設計: 検索エンジンからの流入を見据えたデータ構造の最適化。

##  使用技術

- Backend: Python
- Database: SQLite / PostgreSQL
- Frontend: HTML / CSS / JavaScript
- Others: データ収集用ライブラリ (例: BeautifulSoup, Selenium 等)
